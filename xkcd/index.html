Om de strips op te halen gebruik je de API die wordt aangeboden door xkcd:
xkcd api
Requirements
De applicatie haalt de afbeeldingen op via de xkcd API.
Als je de afbeelding hovered krijg je de tooltip tekst te zien ("alt" attribuut).
Random laat een random bestaande cartoon zien.
Als je voorbij de nieuwste klikt ga je terug naar de nieuwste.
Routing sluit aan op het xkcd nummer (localhost:4200/2484 mapt naar cartoon 2484).
Als de routing niet resolved ga je naar de nieuwste image.
De knoppen werken allemaal! (kijk op de site wat ze doen)
De "permanent" link wordt gemaakt voor je eigen website.
Bonus: de image url voor hotlinking/embedding.
Bonus: Unit testen >_>
